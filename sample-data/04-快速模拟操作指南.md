
### 1.5 提交步骤

```
1. 所有模块上传完成并验证通过后
2. 点击 [提交申请] 按钮
3. 在确认弹窗中点击 [确认提交]
4. 系统生成受理号，格式: CYHS2400001
5. 记录受理号 → 进入"受理审查"阶段
```

```
7. 确认提交 → 系统生成FDA Tracking Number
```

## 常见问题速查

### NMPA 常见问题

| 问题 | 解决方案 |
|------|----------|
| 验证失败: "M1缺少证明性文件" | 确保上传了`02-证明性文件.txt`，PDF中需包含营业执照、生产许可证、GMP证书 |
| 受理号生成后可以修改吗？ | 不可以。受理后仅能提交补正材料，不能修改原始申请 |

### FDA 常见问题

| 问题 | 解决方案 |
|------|----------|
| eCTD验证报错: "US M1 sequence must include form 356h" | 在m1/us/356h目录下放入填写完整的356h表 |
| ESG证书过期了怎么办？ | 模拟环境: 在Settings > Certificates > Renew 点击自动续期即可 |
| DUNS未找到 | 确认格式正确: XX-XXX-XXXX (如 12-345-6789) |

### EMA 常见问题

| 问题 | 解决方案 |
|------|----------|
| 提交被拒: "ERA phase I assessment missing" | 补充上传环境风险评估Phase I报告 (参考EMEA/CHMP/SWP/4447/00) |
| SmPC不符合QRD模板 | 从EMA官网下载最新版QRD SmPC模板，按章节重新排版 |
| PIP未批准就想提MAA | 必须先获得PDCO的PIP批准决定，MAA才能被受理 |

## 模拟通关目标

按以下顺序完成三系统全部模拟操作：

1. **NMPA**: 登录 → 填写申请表 → 上传eCTD → 提交 → 获得受理号
2. **FDA**: 登录ESG → 创建NDA → 填写356h → 上传eCTD → 提交 → 获得Tracking Number
3. **EMA**: 登录Gateway → 创建MAA → 填写信息 → 上传eCTD+EU特有文件 → 提交 → 获得Procedure Number

**预计总用时：** 45-60分钟 (含eCTD文件上传和验证等待)

**模拟评价标准：**
- 三个系统均提交成功 = 通关
- 验证报错数 ≤ 3 = 优秀
- 验证报错数 ≤ 10 = 合格
- 验证报错数 > 10 = 需重新准备eCTD文件
